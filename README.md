### 👋 About Me

I'm a **SAP Customer Checkout (CCO) Plugin Developer** with hands-on experience designing and shipping **production-grade plugins for retail point-of-sale (POS) environments**. I build the logic that runs at the till — promotions, payments, and security — fully integrated into live SAP CCO checkout workflows across multi-branch retail.

- 🧩 Shipped a complete suite of **8 production SAP CCO plugins** + a centralized **Plugin Manager Portal**
- 🔗 Work across the full retail stack: **SAP Business One → SAP CCO → POS**
- 🧰 Core stack: **C# / .NET · REST APIs · SQL Server · Git**
- 🌍 Deliver multilingual checkout apps in **English, Hindi & Urdu**
- 💼 Currently **open to freelance SAP CCO / POS projects**

### 🚀 Selected Work

> These are **production systems** built for live retail operations. The source is proprietary to the employer, so they're described here as case studies rather than public repositories.

**🧩 SAP CCO Plugin Suite — 8 production plugins**
A complete checkout suite working together in live retail across company branches:
`Discount` · `Coupon` · `BOGO` · `FOC` · `Cash Discount` · `Credit Card` · `EFT Automation` · `Security`

**🖥️ SAP CCO Plugin Manager Portal**
A centralized web portal to configure, deploy, update and monitor every CCO plugin across all shop branches from a single interface — cutting per-store maintenance effort.

**🔄 SAP B1 ↔ CCO Discount-Sync Resolution**
Traced and fixed `B1_CUSTOMER_GROUP_MISSING` discount-sync failures by creating the missing customer groups in SAP Business One and correcting the data-replication order to CCO and POS.
